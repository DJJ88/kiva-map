This page is used to track the progress of a Python script whose purpose is to incrementally acquire and process all lender-loan data from Kiva. It goes through each loan in the Kiva data snapshot (there's about 412k of them) and fetches all its lenders from the Kiva API. Then it stores the lenders, loans, and the lender-loan connections into files.

The files are then read by an R script which plots the lenders and loans and draws lines between them to create a Kiva world map visualization. I did this previously with an outdated snapshot, and now that it's been recently updated, I'm updating the visualization.
